# 女性身材穿搭推荐助手（MVP）

一个纯前端 Demo：输入常穿尺码、身高、体重与穿搭场景后，给出推荐尺码和基础穿搭建议。

## 本地运行

```bash
cd tools/outfit-fit-app
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 当前规则（可继续迭代）

- 尺码推荐：综合身高、体重、BMI，并和用户常穿尺码做折中。
- 体型判断：基于 BMI 划分为纤细 / 匀称 / 微曲线 / 丰满。
- 穿搭建议：按场景（通勤、约会、运动）给出模板，并叠加体型建议。
