# Tests Folder Map

Tests are grouped by behavior area.

- `Balance/`: chain-specific balance service tests.
- `Networking/`: network client and HTTP behavior tests.
- `Diagnostics/`: diagnostics data and bundle tests.
- `Planning/`: refresh/scheduling planner tests.
- `Store/`: persistence and secure store tests.
- `Docs/`: testing process/reference docs.

If you add a new test, place it by behavior rather than by layer implementation.
