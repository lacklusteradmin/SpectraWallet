# Views Folder Map

This folder is organized by UI product area.

- `Dashboard/`: portfolio overview, history, and transaction detail screens.
- `WalletFlow/`: wallet setup/import, receive, and send flow screens.
- `Settings/`: settings and token visibility screens.
- `ChainWiki/`: chain knowledge/reference UI.
- `Diagnostics/`: diagnostics and troubleshooting UI.
- `Endpoints/`: endpoint and provider configuration UI.
- `Donations/`: donation and support UI.
- `Staking/`: staking UI.
- `Models/`: UI-facing model and presentation helpers.
- `Support/`: UI support utilities (interop, localization, activity helpers).
- `Resources/`: assets, localization, and static data consumed by Views.

`ContentView.swift` now lives at `Views/App/ContentView.swift`.
